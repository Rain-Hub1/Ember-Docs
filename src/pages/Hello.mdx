# Introdução à Video Platform

A Video Platform é uma aplicação de referência (boilerplate) que demonstra como construir um serviço de upload e streaming de vídeos usando uma arquitetura JAMstack moderna.

O objetivo deste projeto é fornecer uma base sólida e bem documentada para desenvolvedores que desejam criar aplicações semelhantes, combinando serviços gratuitos e escaláveis.

## Arquitetura Principal

A plataforma é desacoplada em três componentes principais, cada um com uma responsabilidade clara:

- **Frontend (GitHub Pages):** Uma interface de usuário estática, construída com HTML, CSS e JavaScript puro para máxima performance.
- **Backend (Render):** Um servidor Node.js/Express que atua como o cérebro da aplicação, gerenciando a lógica de negócio e a comunicação segura com os serviços.
- **Serviços (Supabase):** Um conjunto de ferramentas que fornecem o banco de dados (PostgreSQL) e o armazenamento de arquivos (S3-like Storage).

Este modelo permite que cada parte da aplicação seja desenvolvida, implantada e escalada de forma independente.
